# 🎓 Student Record Management System

A simple **Student Record Management System** built using **Python**. This is a console-based application that allows users to manage student records efficiently using object-oriented programming (OOP).

---

## 📌 Features

- ➕ Add a new student
- 📋 Display all student records
- 🔍 Search for a student by Roll Number
- ✏️ Update student details
- 🗑️ Delete a student record
- 📊 Sort students by Roll Number
- ❌ Exit the application

---

## 🛠️ Technologies Used

- Python 3
- Object-Oriented Programming (OOP)

## ▶️ How to Run

1. Clone the repository.

```bash
git clone https://github.com/your-username/student-record-management-system.git
```

2. Open the project folder.

```bash
cd student-record-management-system
```

3. Run the program.

```bash
python student_record.py
```
